# Personal Todo GraphQL API

This is a personal project focused on learning and implementing GraphQL concepts. The project serves as a hands-on exploration of GraphQL setup and implementation using NestJS framework.

## Project Overview

This API manages a Todo list system using GraphQL, demonstrating:

- Basic GraphQL setup with NestJS
- CRUD operations for Todo items
- GraphQL schema definition and resolvers
- Data persistence and management

## Purpose

This project was created for personal development and learning purposes, specifically to:

- Understand GraphQL fundamentals
- Learn NestJS integration with GraphQL
- Practice TypeScript in a GraphQL context
- Explore GraphQL schema design

## Technology Stack

- NestJS
- GraphQL
- TypeScript
- Node.js

## Getting Started

### Installation

```bash
npm install
```

### Running the Application

```bash
# Start the application
npm run start

# Development mode (with watch)
npm run dev

# Production mode
npm run prod
```

### Testing

```bash
# Run unit tests
npm run test

# Run e2e tests
npm run test:e2e

# Run test coverage
npm run test:cov
```
